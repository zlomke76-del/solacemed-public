# SolaceMed.ai Public Architecture

SolaceMed.ai frames healthcare AI governance around the transition from recommendation to operational consequence.

## Public Layers

1. Reality Contact — is the state still grounded?
2. Runtime Admissibility — is action still legitimate now?
3. Consequence Boundary — is the system crossing into clinical consequence?
4. Escalation Integrity — is human review required and available?

## Public Boundary

This public repository describes concepts, public-facing surfaces, and demo-ready framing. Production clinical infrastructure, patient data, authenticated workflows, internal orchestration, and deployment secrets remain outside this public repository.
